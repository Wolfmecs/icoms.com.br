ERP

Sistemas ERP (Enterprise Resource Planning) são sistemas de informações que integram pacotes de dados e processos de um software comercial com a finalidade de dar suporte à maioria das operações da empresa. Esse pacote permite a uma companhia automatizar e integrar a maioria de seus processos internos e externos de transações, compartilhar práticas e dados comuns através de toda a empresa e produzir o acesso à informação em um ambiente de tempo real.
* A figura ao lado representa uma estrutura típica do sistema ERP.

Globe

Uma das maiores vantagens de nosso sistema é a abertura da empresa para que os clientes possam requisitar novas funcionalidades que ajudem a integralização total do pacote ERP com os detalhes específicos de cada empresa, seja ela uma industria, uma representação, uma distribuidora ou uma prestadora de serviços em geral. Dada a integração do sistema para multiplas empresas, podemos então controlar qualquer um dos tipos de organizações outrora citados em uma só base de dados, podendo até mesmo integrar os dados entre elas, conforme a necessidade do cliente.
Características gerais do sistema Globe ERP:
  • Aplicação Cliente/Servidor voltada para operação online;
  • Grande Flexibilidade;
  • Múltiplas Empresas;
  • Controle de Acessos e Ações;
  • Ferramenta para Criação de Relatório Dinâmico;
  • Linguagem Script para Composição de Preços;
  • Banco de Dados Relacional (sem Custo de Propriedade) FireBird;
  • Acesso nativo ao BD via TCP/IP.
